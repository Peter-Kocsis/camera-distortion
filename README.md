# Camera distorsion

A module for handling camera distorsion. Originally developed for fisheye-removal.

### Features
* Camera calibration: Determine the intrinsic camera parameters
* Image undistorsion
* Video undistorsion

### Available camera parameters
* GoPro Hero 7 Silver
